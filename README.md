# 📈 WillyStock

관심 주식의 **관심가**와 **목표가**를 등록하고, 가격 도달 시 **푸시 알림**을 받는 PWA 앱입니다.
iOS와 Android에서 홈 화면에 추가하면 네이티브 앱처럼 작동합니다.

## 실행 방법

```bash
npm install
npm start
# 브라우저에서 http://localhost:3000 접속
```

개발 모드 (자동 재시작):
```bash
npm run dev
```

## 주요 기능

- 🔍 **종목 검색**: 한국(KOSPI/KOSDAQ) + 미국 주식 검색
- 🔔 **관심가 알림**: 설정 가격 이하 도달 시 푸시 알림
- 🎯 **목표가 알림**: 설정 가격 이상 도달 시 푸시 알림
- 📲 **PWA 설치**: Android/iOS 홈 화면 추가
- 🌙 **다크/라이트 모드**
- 📶 **오프라인 지원**: 마지막 데이터 캐싱

## iOS 설치 방법

1. Safari에서 앱 접속
2. 하단 공유 버튼(□↑) 탭
3. **홈 화면에 추가** 선택
4. iOS 16.4 이상에서 푸시 알림 지원

## 가격 모니터링

- 2분마다 Yahoo Finance에서 가격 조회
- 관심가/목표가 도달 시 푸시 알림 발송
- 4시간 이내 같은 알림 중복 방지
- 가격이 임계값에서 벗어나면 알림 초기화

## 주식 티커 형식

| 거래소 | 예시 |
|--------|------|
| 한국 KOSPI | `005930.KS` (삼성전자) |
| 한국 KOSDAQ | `035720.KQ` (카카오) |
| 미국 NASDAQ | `AAPL`, `TSLA` |
| 미국 NYSE | `MSFT`, `AMZN` |
